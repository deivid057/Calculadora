# Calculadora simples feita com Python!

### Especifique qual operação deseja e depois os elementos. ;)
---
## Exemplo

As opções são:

*`1- Soma`*

*`2- Subtração`*

*`3- Multiplicação`*

*`4- Divisão`*

*`5- Sair da Calculadora`*

Se vc escolher o número 1 ele vai pedir que vc digite os elementos da soma!
